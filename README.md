This repository contains the scripts to reproduce the figures in the manuscript Denaud at al. Genome Biology, 2025

Please, refer to each of the following folders to perform the analysis of the corresponding genomic assay:
- CnR_analysis
- HiC_analysis
- RNAseq_analysis

## Contributions ##
The code in this repository has been developed at the [Cavalli Lab](https://www.igh.cnrs.fr/en/research/departments/genome-dynamics/chromatin-and-cell-biology) with the contributions of Marco Di Stefano, Gonzalo Sabaris, and Giorgio L. Papadopoulos.
